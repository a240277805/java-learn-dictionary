

# 缓存实施情况

**redis key 规则定义**

定义redis key 规则方便 定位，拆分 或 附加的业务逻辑处理 。

比如： mq 聚合 缓存操作时统一处理

规则:**{项目}:{服务名}:{业务模块}:{type}:{缓存接口}**

 **type: **

* method-服务方法
* dictionary-字典**
